- ### 2.1.1 单位属性

|     属性     |                             说明                             |
| :----------: | :----------------------------------------------------------: |
|     编号     |                         unit`_`序号                          |
|     类型     |                         用于分类单位                         |
|     名称     |                              -                               |
|     介绍     |                              -                               |
|  等级Level   |               单位总等级 = 单位等级 + 玩家等级               |
|  稀有度rare  |                普通N，稀有R，史诗SR，传说SSR                 |
|    生命值    |                   当生命值为0时，单位死亡                    |
| 护盾值shield |         可以用来抵消伤害，具体抵消的伤害类型查看1.4          |
|   物伤减免   | 按百分比减免所受到的物理伤害，比如当物伤减免为30%时，100的物理伤害最终只会造成70点物伤。<br />上限[-100%, 100%] |
|   法伤减免   |       按百分比减免所受到的魔法伤害。上限[-100%, 100%]        |
|     物攻     |                 一次普通攻击所造成的物理伤害                 |
|     法强     |         法强会倍化一个技能的魔法伤害、治疗效果、护盾         |
|    暴击率    |    指在一次普通攻击[非真实伤害]中发生暴击的概率。上限100%    |
|   暴击效果   | 单位的基础暴击效果固定为原物理伤害的150%，额外暴击效果在这个基础上相加。比如额外暴击效果为50%<br />则此次暴击伤害为 物理攻击×（150% + 50%） |
|    闪避率    | 只能闪避物理伤害。当闪避判定成功时，将此次伤害的最终伤害修改为0 |
|     攻速     |                  两次普通攻击之间的间隔时间                  |
|    法力值    |               当法力值积满时，自动释放一次技能               |
|   生命偷取   | 对敌人造成物理伤害时，会回复相当于伤害量一定比例的生命值。比如敌人受到的最终伤害为100，生命偷取为10%，<br />则此次攻击可回复10点生命值 |
|   攻击回蓝   |              每造成一次物理伤害，回复5点法力值               |
|   承伤回蓝   | 每受到一次伤害，回复伤害值×0.05的法力值，数值向下取整。单次承伤回蓝上限50 |
|   普攻目标   |                              -                               |
|   主动技能   |                  法力值积满时，释放主动技能                  |
|   被动技能   |                 单位初始入场后初始化被动技能                 |
|     立绘     |                              -                               |
