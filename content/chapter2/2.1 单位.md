- ### 2.1.1 基础属性

|   属性   |                             说明                             |
| :------: | :----------------------------------------------------------: |
|   编号   |                       objective`_`序号                       |
|   类型   |                         用于分类单位                         |
|   名称   |                              -                               |
|   介绍   |                              -                               |
|   等级   |                           三个等级                           |
|  生命值  |                   当生命值为0时，单位死亡                    |
| 物伤减免 | 按百分比减免所受到的物理伤害，比如当物伤减免为30%时，100的物理伤害最终只会造成70点物伤。<br />上限[-100%, 100%] |
| 法伤减免 |       按百分比减免所受到的魔法伤害。上限[-100%, 100%]        |
|   物攻   |                 一次普通攻击所造成的物理伤害                 |
|   法强   |         法强会倍化一个技能的魔法伤害、治疗效果、护盾         |
|  暴击率  |    指在一次普通攻击[非真实伤害]中发生暴击的概率。上限100%    |
| 暴击效果 | 指发生暴击时造成的伤害与原伤害的比值。比如暴击效果为250%，则一次暴击造成的伤害是原来的3.5倍。<br />额外附加的暴击效果在原基础上相加，比如遗物效果增加40%的暴击效果，则最终暴击效果为290% |
|  闪避率  | 只能闪避物理伤害。当闪避判定成功时，将此次伤害的最终伤害修改为0 |
|   攻速   |                  两次普通攻击之间的间隔时间                  |
|  法力值  |               当法力值积满时，自动释放一次技能               |
| 生命偷取 | 对敌人造成物理伤害时，会回复相当于伤害量一定比例的生命值。比如敌人受到的最终伤害为100，生命偷取为10%，<br />则此次攻击可回复10点生命值 |
| 攻击回蓝 |              每进行一次普通攻击，回复5点法力值               |
| 承伤回蓝 | 当生命值为生命上限的100%/80%/60%/40%/0%/时(区间)，**每受到一次伤害**，回复5/10/15/20点法力值 |
| 普攻目标 |                              -                               |
|   技能   |                              -                               |
| 定向升级 |                              -                               |
| 升级描述 |        关于定向升级效果的文字描述，方便界面需要时调用        |
| 附加属性 | 用于记录通过事件获得的额外属性加成，在战斗开始时，以单个buff的展现形式启用所有加成 |
|   立绘   |                              -                               |

- ### 2.1.2 卡组分组<div id="fenlei">

  - 怪物卡组 [ 所有包含 单位.类型 = "monster" ] 
    - 1级怪物卡组 [ 所有包含 单位.类型 = "monster" && 单位.等级 = "1" ]
    - 2级怪物卡组 [ 所有包含 单位.类型 = "monster" && 单位.等级 = "2" ]
    - BOSS怪物卡组 [ 所有包含 单位.类型 = "monster" && 单位.等级 = "3" ]
    - “被污染的精灵之森”怪物卡组 [ 单位.编号 = "objective_001" ~ 单位.编号 = "objective_009" ]
  - 装备卡组 [ 所有包含 单位.类型 = "equipment" ] 
  - 随从卡组 [ 所有包含 单位.类型 = "attendant" ] 
  - 英雄卡组 [ 所有包含 单位.类型 = "hero" ]
