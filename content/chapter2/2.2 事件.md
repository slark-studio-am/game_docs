### 2.2.1 关卡配置<div id="226">

|      字段名      |         字段          |                            值说明                            |  值示例  |
| :--------------: | :-------------------: | :----------------------------------------------------------: | :------: |
|     关卡编号     |       stage_id        |                              -                               | stage_01 |
|   关卡初始奖励   |  list_initial_reward  |          每关在战备阶段开始时自动赠与玩家的卡牌list          |    -     |
|    关卡定期数    |   list_regular_num    |         间隔多少隔关卡后会获得额外奖励，初始关卡为1          |    3     |
| 关卡定期额外奖励 |  list_regular_reward  | 根据关卡间隔数，每隔X关卡在战备阶段开始时自动赠与玩家的卡牌list |    -     |
|     敌方阵容     |    list_enemy_card    |               存放本关卡敌方阵容的单位卡牌list               |    -     |
|     我方阵容     |     list_our_card     |                  存放我方阵容的单位卡牌list                  |    -     |
|   初始怪物等级   | monster_initial_level |               游戏开始时，第一关敌方单位的等级               |    1     |
|   怪物成长等级   | monster_growing_level |                 每过1关，敌方单位的成长等级                  |    1     |



### 2.2.2 事件卡模板 table_event_template<div id="222">

|  字段名  |         字段         |               值说明               |               值示例               |
| :------: | :------------------: | :--------------------------------: | :--------------------------------: |
| 事件编号 |       event_id       |                 -                  |              event_04              |
|  事件名  |      event_name      |             事件的名称             |             祈愿：神器             |
| 事件说明 |    event_details     |         描述事件行为的说明         | 从3个所展示的神器中挑选1个加入手牌 |
| 事件立绘 |     event_image      |             事件的图片             |                 -                  |
| 选中个数 | event_count_selected | 事件卡被使用时有几个可供选择的卡牌 |                 1                  |
| 待选个数 |  event_count_select  |          需要选中几个卡牌          |                 3                  |
| 卡牌来源 |     event_source     |          被挑选卡牌的来源          |      2.3神器 / list_artifact       |

[![2buYBn.png](https://z3.ax1x.com/2021/06/15/2buYBn.png)](https://imgtu.com/i/2buYBn)

| 序号 |                            内容                            |
| :--: | :--------------------------------------------------------: |
|  1   |   根据 event_count_select 和 event_source来展示卡牌个数    |
|  2   |               此处显示事件说明 event_details               |
|  3   | 当所选的卡牌数量不满足event_count_selected时，按键无法点击 |
