# 产业元素

#### 1. 产业&产品

游戏中的产业分为：

- **基础产业**：专门生产原材料的产业，无需上游产业的支持，没有额外的交互界面

| 产业类型 |       主产出       |   副产品   |
| :------: | :----------------: | :--------: |
|   农场   | 玉米、大头菜、鸡肉 |     无     |
|   牧场   |        牛肉        | 肥料、皮革 |
|   矿场   |     木材、铁矿     |    煤炭    |

- **中级产业**：对原材料进行加工制作出成品，需要上游产业的支持，没有额外的交互界面

| 产业类型 |  上游产业  |     产品     |
| :------: | :--------: | :----------: |
|  铁匠铺  |    矿场    | 农具、冷兵器 |
|  酿酒厂  |    农场    |     啤酒     |
|  成衣店  |    牧场    |     套装     |
|  食品厂  | 农场、牧场 |  炸鸡、牛排  |

- **高级产业**：有特殊功能的建筑，具有交互界面，需要多个上游产业的支持

| 产业类型 |   上游产业   | 说明 |
| :------: | :----------: | :--: |
|   酒馆   | 酒厂、食品厂 |      |
