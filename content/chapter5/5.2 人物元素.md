# 人物元素

人物作为一种独立的组件，可以被绑定在实体上。

### 1.1 人物附加属性

|      属性      | 数据类型 |           说明           |
| :------------: | :------: | :----------------------: |
|     生命值     |   int    |  健康值为0时，人物死亡   |
|     疲劳值     |   int    | 用于衡量人物是否需要进食 |
| 疲劳值回复速率 | float(2) |                          |
| 疲劳值回复类型 |  string  |                          |
|    移动速度    |   int    |                          |
|    视线角度    |   int    |                          |
|    视线距离    |   int    |                          |
