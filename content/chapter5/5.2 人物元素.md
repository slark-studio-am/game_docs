### AI人物属性

|  属性  |                  说明                  |
| :----: | :------------------------------------: |
| 健康值 |                 生命值                 |
| 饱食度 |                                        |
| 营养度 | 营养度维持高位会更有机会获得新的好天赋 |

- 饱食度低于40%时，人物处于饥饿状态开始寻找食物进食
- 饱食度维持0 xx时间后，开始扣除健康值
- 饱食度维持100 xx时间后，营养度开始缓慢增长
