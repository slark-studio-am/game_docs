#### ⑨的各类属性说明

|    属性    |                           内容                           |
| :--------: | :------------------------------------------------------: |
| 物理攻击力 |                 1次普通攻击造成的伤害量                  |

|  承伤回魔  |                1次受到伤害时回复的法力值                 |

### 4.4 治疗室 Bottom/Build_training下半<div id="44">

#### （1）组件说明

[![WNjlM6.png](https://z3.ax1x.com/2021/07/20/WNjlM6.png)](https://imgtu.com/i/WNjlM6)

| 序号 |        标题        |                             内容                             |
| :--: | :----------------: | :----------------------------------------------------------: |
|  1   | 【文字】练功房标题 |       3个练功房都是一致的，其他信息参照上面的三个房间        |
|  2   |  【按键】取消按键  | 当英雄被拉入练功房但是尚未选择要学习的技能时，可以通过此按键离开房间 |
|  3   |  【按键】学习花费  | 学习技能所要花费的灵魂，单个房间单次只能学习一个技能，即3选1选中了技能后，剩余的两个技能将消失 |

注：

（1）英雄拖入练功房时，不会消耗灵魂，只有选择了技能后才会消耗对应数量的灵魂

（2）退出英雄除了按取消按键之外还有一种情况，如果上一个英雄被拖入练功房但是没选择学习哪个技能，那么当下一个英雄被拖入这个房间或拖入另外的房间时，上一个英雄重新返回队伍

（3）英雄在尚未确定学习哪个技能的时候仍然是处于“待命”状态

（4）玩家在“我的队伍”中点击或触摸英雄时，③的选技能区就会更新目前点击或触摸英雄的3个可选技能；当玩家将某一个英雄拖动到练功房之后，练功房便只会显示该英雄的可选技能，直到该英雄被通过（2）的两种方式退出房间

### 4.6 其他临时数据和补充说明<div id="46">

|       项目       |        内容        | 灵魂数 |
| :--------------: | :----------------: | :----: |
| 每回合可用灵魂数 | 在战备开始阶段回满 |   8    |

（1）遗物奖励：无需开宝箱，直接将奖励给予玩家，奖励目前包含遗物和额外灵魂，在关卡设置中配置第几关送和送的内容。直接用提示的方式，时机就在战斗结束后，神器2选1选择完后显示奖励灵魂的提示，奖励灵魂提示停留展示2s后执行正常的阶段切换

![image-20210721204921086](https://i.loli.net/2021/07/21/9uCxXWvkawi7qpo.png)

（2）初始：随机送3个英雄到我的队伍。

（3）解雇补偿：返还50%所投入的灵魂数。投入的灵魂值得是【该英雄的雇佣价格】+【该英雄身上所有附加被动技能的学习花费】

（4）练功房：取消被动技能的学习时间，只需要消耗灵魂。删除掉英雄的“练功”状态。提高学习被动技能的花费，调高怪物的基础伤害，让需要回血变得频繁，英雄设计上更偏向职业特色技能，让刷酒馆更有意义一点。

（5）人物详情中的属性：属性数值无变化-白色；比原始数值高-绿色；比原始数值低-红色
