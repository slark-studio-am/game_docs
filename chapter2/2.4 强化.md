### 2.4.1 属性模板&分组<div id="241">

##### （1）属性模板

|    字段名    |        字段        |                            值说明                            |       值示例       |
| :----------: | :----------------: | :----------------------------------------------------------: | :----------------: |
| 强化效果编号 |   strengthen_id    |               描述该强化效果在系统中的唯一标识               |   strengthen_01    |
| 强化效果名称 |  strengthen_name   |                        强化效果的名称                        |      攻速强化      |
| 强化效果类别 |  strengthen_type   | 基础base：同一个list下的单位可用<br />类category-XX：同一个list下的单位可用<br />专属Exclusive-XX：XX单位专用 | Exclusive-unit_001 |
|    稀有度    |  strengthen_rare   |                    描述强化效果的出现概率                    |         1          |
|   是否重复   | strengthen_repeat  |          描述在同一个list中，抽取出的效果是否被放回          |         是         |
|   强化等级   |  strengthen_level  |                  强化效果的等级，和强度相关                  |         1          |
| 强化效果说明 | strengthen_details |                      描述强化效果的说明                      |         -          |
| 强化效果数值 |  strengthen_value  |                 用于描述强化效果说明中的数值                 |         -          |
| 强化效果原画 |  strengthen_image  |                      强化效果的卡面原画                      |         -          |

##### （2）分组

|              分组编号              |         分组说明         |                           组内卡牌                           |
| :--------------------------------: | :----------------------: | :----------------------------------------------------------: |
|        list_strengthen_base        |       基础强化效果       |                    strengthen_type='base'                    |
|  list_strengthen_category_warrior  |      战士类强化效果      |              strengthen_type='category_warrior'              |
|   list_strengthen_category_mage    |      法师类强化效果      |               strengthen_type='category_mage'                |
| list_strengthen_category_physical  |      物攻类强化效果      |             strengthen_type='category_physical'              |
| list_strengthen_Exclusive_unit_001 | 圣廷骑士团首领帕拉丁专属 | **strengthen_type='Exclusive_unit_001**' + <br />**list_strengthen_base** + <br />**list_strengthen_category_warrior** |
| list_strengthen_Exclusive_unit_002 |     裂颅者斯卡西专属     | **strengthen_type='Exclusive_unit_002**' + <br />**list_strengthen_base** + <br />**list_strengthen_category_warrior** |
| list_strengthen_Exclusive_unit_002 |     云霄巨人明尧专属     | **strengthen_type='Exclusive_unit_003**' + <br />**list_strengthen_base** + <br />**list_strengthen_category_warrior** |
