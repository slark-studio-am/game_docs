### 3.1.1 技能设计<div id="231">

#### 3.1.1.1 技能词缀

| 词缀 |         词条说明         |                   说明                   |
| :--: | :----------------------: | :--------------------------------------: |
| 登场 |   从手牌被召唤到战场时   | 单位卡从手牌区到战场，等同于炉石**战吼** |
| 进攻 |      进行普通攻击时      | 一次完整的普通攻击链条结束后开始执行效果 |
| 余威 |      发动主动技能时      | 一次完整的主动技能链条结束后开始执行效果 |
| 刺激 | 每场战斗第一次受到伤害时 |            有生命值的变化才算            |
| 激怒 |        受到伤害时        |            有生命值的变化才算            |
| 备战 |      战备阶段开始时      |    战斗结束后，回到战备阶段的刚开始时    |
| 濒死 |    当生命值低于30%时     |   每次低于这个血量都触发，可以反复触发   |
| 退场 |        战斗结束时        |    每一场战斗结束后单位仍然存活时触发    |
| 死兆 |       当单位死亡时       |         从战场被移动到墓地时触发         |
| 先声 |     当一场战斗开始时     |              按下战斗键以后              |
| 受击 |        受到攻击时        |        不管有没有受到伤害都算触发        |

#### 3.1.1.2 技能描述中的目标说明

|  目标  |      指代单位      |
| :----: | :----------------: |
|  单位  |  包含英雄和召唤物  |
|  英雄  |         -          |
| 召唤物 |         -          |
|  自己  | 发动技能的宿主单位 |

#### 3.1.1.3 技能描述中的简略说明

|      描述       |                 说明                 |
| :-------------: | :----------------------------------: |
|  生命值少于30%  | 指显示生命值低于显示最大生命值的30%  |
|   驱散DEBUFF    |          将buff的计时器清零          |
|  已损失生命值   |   指显示生命值与显示最大生命值的差   |
| 已损失10%生命值 |   指显示生命值与显示最大生命值的比   |
|  生命比例最低   | 指显示生命值与显示最大生命值的比最低 |
